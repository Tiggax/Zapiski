$p[1,...,m];t[1,...,n]\in \Sigma$

$Def:$ 
Sreda(p,t):
```
Sreda(p,t):
	n=|t|; m=|p|;
	for i=1...n-m:
		j=1
		while(j<=m)&&(t[i+j-1]=p[j]):
			j=j+1
		if j>m:
			r=r*"i";
	return r
```
