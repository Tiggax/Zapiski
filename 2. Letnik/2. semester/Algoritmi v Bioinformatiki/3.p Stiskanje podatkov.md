## Primer
$w=ACAGAGAAA...$ -> vsaka **liha** črka je $A$, vsaka **soda** je $\in\{A,T,C,G\}$
$\#A=\#T=\#C=\#G$-> so enako pogosti

### z enako dolgim
$$\begin{align}
koda=&C_{A}\to 00\\
&C_{T}\to 11\\
&C_{C}\to 01\\
&C_{G}\to 10\\
\end{align}$$
```
def code(w,c[]):
	n=|w|;r=1
	for i=1...n:
		r=r*c[w[i]]
	return r
```
> $O(n)$
```
def decode(w,d[]):
	n=|w|/2; r=1,j=1
	for i=1...n:
		r=r*d[w[j],w[j+1]]
		j=j+2
	return r
```
> $O(n)$

**RDEČE** so Borrows Wheller Transform
$BWT(w)\to u:O(n)$

$BWT^{-1}(u):$
$i$-ta pojavitev v $BWT(w)$-ju je ista kot v $i$-ta pojavitev v $SA$
v ta namen definiramo:

$LF[i]$: i-ta črka v $BWT$ je $LF[i]$-ta črka v $SA$

$$\begin{align}
&BWT: &A& &G& &G& &C& &\$& &A& &A& \\
&LF[i]: &2& &6& &7& &5& &1& &3& &4&\\
\\
&SA: &\$& &A& &A& &A& &C& &G& &G&\\
\end{align}$$
iz $BWT(n)$ in LF -> $w$ v $O(n)$
